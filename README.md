Aurora Records – IT Support Simulation Project

Overview
Aurora Records is a simulated record label IT environment created to develop and demonstrate practical IT Support and Service Desk skills. The project is designed to replicate a real-world workplace IT department, including users, systems, and common technical issues.
This project is part of my ongoing learning in IT Support, focusing on Windows environments, troubleshooting, service desk workflows, and basic system administration.

 Project Goals
* Simulate a real IT support environment for a fictional company
* Practice troubleshooting common IT issues
* Develop understanding of service desk ticket workflows
* Build confidence using Windows Server and Windows client systems
* Document technical processes clearly and professionally

Simulated Company Structure
Aurora Records includes multiple departments to reflect a real workplace environment:
* Production Team
* Marketing Department
* Finance Department
* Administration
Each department has users who may experience IT-related issues such as login problems, software errors, or connectivity issues.

Technologies & Tools Used
* Windows Server (Home Lab)
* Windows 10/11
* Ubuntu (Linux basics)
* Active Directory (learning/fundamentals)
* Microsoft 365 (conceptual knowledge)
* VirtualBox (virtualisation platform)
* Service Desk Simulation (self-learning)
* GitHub for documentation

Project Structure
This repository includes:
* IT support tickets (simulated incidents and resolutions)
* User and department setup documentation
* System and network configuration notes
* Troubleshooting guides and knowledge base articles

Example Scenarios
This project simulates real IT support issues such as:
* Password resets and account lockouts
* WiFi and network connectivity issues
* Email and Microsoft Outlook problems
* Printer setup and driver issues
* Software installation and configuration problems

What I Am Learning
Through this project I am developing:
* First-line IT support and troubleshooting skills
* Understanding of service desk workflows (ticket logging, prioritisation, resolution)
* Windows Server and Active Directory fundamentals
* Clear technical documentation and communication skills
* Problem-solving in real-world IT scenario

Future Improvements
* Expand Active Directory structure with more advanced configurations
* Add Group Policy Object (GPO) simulations
* Introduce more complex network troubleshooting scenarios
* Build a full ticketing lifecycle system (New → Resolved)
* Add PowerShell scripting basics
  
Purpose
This project is part of my journey into IT Support and is intended to demonstrate practical learning, technical curiosity, and readiness for an IT apprenticeship or entry-level support role.
