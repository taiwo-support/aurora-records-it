# Tickets System - T-Money Records

This folder tracks all internal tasks and operations for the label.

## How it works:
- Each ticket is a separate markdown file
- Tickets are numbered (ticket-001, ticket-002, etc.)
- All work must be tracked through tickets

## Status meanings:
- Open = Not started
- In Progress = Being worked on
- Done = Completed but not closed
- Closed = Fully finished and archived

## Rules:
- Never delete tickets
- Always update status before committing changes
- Use the template for consistency# Tickets System - T-Money Records
